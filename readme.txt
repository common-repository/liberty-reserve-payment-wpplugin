== Installation ==

1. Upload Directory `liberty-reserve-payment-wpplugin` to the `/wp-content/plugins/` directory

2. Activate the plugin through the 'Plugins' menu in WordPress

3. input the shortcode <code>[libertyreserve]</code> on your page and Publish !!

PERFECT!
